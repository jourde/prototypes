# Moodle Tool Guide – Simplified list

## Resources – Inform

| Tool | Description |
|---|---|
| **Text and media area (label)** | Insert text and multimedia |
| **File** | Upload a single file |
| **Folder** | Upload several files in a folder |
| **Page** | Create a webpage |
| **Book** | Create a series of pages with chapters and sub-chapters |
| **URL** | Insert link to a web page |

## Activities – Inform and Evaluate

| Tool | Description |
|---|---|
| **Forum** | Promote discussion: debates, reporting, role-playing, news analysis, lists of ideas, etc. |
| **Assignment** | Use to collect, assess and provide feedback on assignments |
| **Quiz** | Use to assess learning, formative or summative. Evaluation automatic |
| **Lesson** | Use to flexibly present branched info, with testing |
| **Chat** | Hold real-time text chat discussions |
| **Workshop** | Collect, assess and generate peer review of student work |

## Activities – Share Data

| Tool | Description |
|---|---|
| **Database** | Allow pupils to collect, share and search created artifacts |
| **Glossary** | Gather resources or collect information |
| **Wiki** | Enable the creation of collaborative pages, like Wikipedia |

## Activities – Identify Trends

| Tool | Description |
|---|---|
| **Choice** | Enable pupils to vote, to choose subjects; in order to get a trend |
| **Feedback** | Gather data from pupils on any topic |
| **Survey** | Gather data from pupils about teaching of the course |
