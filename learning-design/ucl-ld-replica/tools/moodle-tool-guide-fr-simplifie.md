# Guide des outils Moodle – Liste simplifiée

## Les ressources – Informer

| Outil | Description |
|---|---|
| **Zone texte et média (étiquette)** | Insérer du texte et des médias |
| **Fichier** | Déposer un simple fichier |
| **Dossier** | Déposer plusieurs fichiers dans un répertoire |
| **Page** | Insérer le contenu d'une page Internet |
| **Livre** | Créer un texte de plusieurs pages avec chapitres et sous-chapitres |
| **URL** | Insérer un lien vers une page Internet |

## Les activités – Informer et évaluer

| Outil | Description |
|---|---|
| **Forum** | Favoriser les discussions : débats, rapports, jeux de rôles, analyse d'info, listes d'idées, etc. |
| **Devoir** | Proposer une tâche à réaliser, récolter les résultats et les évaluer |
| **Test (quiz)** | Évaluer au moyen de questions et afficher les réponses correctes avec feedbacks. Évaluation automatique |
| **Leçon** | Transmettre des infos de façon flexible, avec des questions de validation |
| **Chat** | Discuter en temps réel |
| **Atelier** | Collecter, évaluer et faire évaluer les travaux des étudiants par leurs pairs |

## Les activités – Mettre en commun des données

| Outil | Description |
|---|---|
| **Base de données** | Collecter, partager et rechercher des artefacts créés |
| **Glossaire** | Rassembler des ressources ou collecter des informations |
| **Wiki** | Permettre la création de pages en collaboration, comme Wikipedia |

## Les activités – Faire ressortir des tendances

| Outil | Description |
|---|---|
| **Sondage** | Permettre de voter, de choisir des sujets ; pour obtenir une tendance |
| **Feedback** | Recueillir des données auprès des élèves, sur n'importe quel sujet |
| **Consultation** | Recueillir des feedbacks sur son enseignement |
