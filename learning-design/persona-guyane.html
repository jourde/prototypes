Guide d'utilisation

1. Objectif
Ce canevas sert à décrire des profils types (personas) afin de concevoir des formations réellement adaptées aux besoins, motivations, contraintes et contextes d’exercice.

2. Gérer plusieurs personas
La barre de personas, juste sous l’en-tête, permet de travailler sur plusieurs profils dans la même session :

  • ◀ ▶ : naviguer entre les personas. Le formulaire bascule automatiquement et l’état du persona quitté est préservé en mémoire.
  • person_add Nouveau : crée un persona vierge.
  • content_copy Dupliquer : duplique le persona courant (utile pour créer des variantes).
  • person_remove Supprimer : supprime le persona courant après confirmation (inactif s’il n’en reste qu’un).

3. Basculer les vues
Deux interrupteurs pilotent l’affichage pour le persona courant :

  • Standard / Guyane : active les items spécifiques au contexte guyanais (isolement, climat, multilinguisme, etc.).
  • Suggestions : affiche ou masque les listes de cases à cocher dans tous les blocs.

4. Renseigner un persona

  • Texte libre : décrivez les éléments qualitatifs (exemples, anecdotes, précisions de contexte).
  • Cases à cocher : activez l’interrupteur Suggestions pour faire apparaître les options prédéfinies, puis cochez ce qui correspond au profil.

5. Import & Export
Le bloc IMPORT & EXPORT couvre tous les personas de la session :

  • upload_file Charger : recharge un fichier .json précédemment sauvegardé (un ou plusieurs personas).
  • save Sauvegarder : exporte tous les personas dans un fichier .json pour reprendre la session plus tard.
  • visibility Synthèse et export : affiche tous les profils dans une fenêtre, avec des boutons pour copier en texte brut, copier en Markdown, télécharger un fichier .md ou télécharger un fichier Word (.doc).

6. Améliorations

  • list_alt Canevas complet : exporte un fichier Markdown exhaustif listant toutes les options (cochées ou non) pour chaque persona — utile pour documenter ou réviser l’ensemble du canevas.
  • feedback Proposer une amélioration : soumettre un retour sur le contenu ou l’ergonomie. Les suggestions sont stockées localement et exportables.

Note
Cet outil est un prototype non officiel. Toutes les données restent dans votre navigateur, rien n’est envoyé en ligne.

À propos

Cet outil permet de modéliser les besoins, motivations et contraintes des personnels d'éducation afin de concevoir des formations adaptées.

Ce document est un prototype non officiel, partagé à des fins de collecte de suggestions d’amélioration.

Traitement local : toutes les données restent sur votre navigateur, rien n'est envoyé en ligne.
